# 🚀 ZYPHER

O ZYPHER é um sistema inteligente de monitoramento de impressoras térmicas, desenvolvido para acompanhar em tempo real o funcionamento dos dispositivos, fornecendo informações precisas, relatórios automatizados e controle centralizado.

Atualmente, o sistema está em utilização dentro da EUROFARMA, auxiliando no monitoramento operacional e na gestão de impressoras em múltiplas filiais.

---

# 📌 Funcionalidades

✅ Monitoramento em tempo real  
✅ Exibição de status e erros das impressoras  
✅ Inventário automatizado  
✅ Relatórios completos para auditoria e controle  
✅ Atualizações em tempo real sem recarregar a página  
✅ Interface web responsiva e intuitiva  
✅ Controle centralizado de múltiplas filiais  

---

# 🛠️ Tecnologias Utilizadas

- Python
- Flask
- Flask-SocketIO
- HTML5
- CSS3
- JavaScript
- Bootstrap
- SQLite / MySQL
- API Zebra / SNMP

---

# 🧠 Arquitetura do Sistema

O sistema utiliza uma arquitetura cliente-servidor, onde:

- O backend em Python coleta os dados das impressoras
- O Flask gerencia a aplicação web
- O Socket.IO envia eventos em tempo real ao frontend
- O frontend atualiza os dados dinamicamente
